# Part 2: WebApp Layer Edition

SVGMap has the unique feature of not only allowing simple static SVGMap content as layers, but also enabling the use of [the WebApp Layer mechanism (Layers as Web Apps: LaWA)](https://www.svgmap.org/wiki/index.php?title=%E8%A7%A3%E8%AA%AC%E6%9B%B8#.E3.82.A6.E3.82.A7.E3.83.96.E3.82.A2.E3.83.97.E3.83.AA.E3.82.B1.E3.83.BC.E3.82.B7.E3.83.A7.E3.83.B3.E3.81.AB.E3.82.88.E3.82.8B.E5.8B.95.E7.9A.84.E3.81.AA.E5.9C.B0.E5.9B.B3.E3.83.AC.E3.82.A4.E3.83.A4.E3.83.BC.E3.81.A8.E3.80.81.E3.81.9D.E3.81.AE.E3.83.8F.E3.82.A4.E3.83.91.E3.83.BC.E3.83.AC.E3.82.A4.E3.83.A4.E3.83.AA.E3.83.B3.E3.82.B0), which allows you to dynamically control and modify the SVGMap content (DOM) of a layer using JavaScript code by associating it with a web application. ([What is a Web Application](https://ja.wikipedia.org/wiki/%E3%82%A6%E3%82%A7%E3%83%96%E3%82%A2%E3%83%97%E3%83%AA%E3%82%B1%E3%83%BC%E3%82%B7%E3%83%A7%E3%83%B3)?)

This mechanism essentially does not affect other layers ([encapsulation](https://ja.wikipedia.org/wiki/%E3%82%AB%E3%83%97%E3%82%BB%E3%83%AB%E5%8C%96)), thus preventing the application from [becoming spaghetti code](https://ja.wikipedia.org/wiki/%E3%82%B9%E3%83%91%E3%82%B2%E3%83%86%E3%82%A3%E3%83%97%E3%83%AD%E3%82%B0%E3%83%A9%E3%83%A0).

The WebApp Layer section involves coding a WebApp that utilizes this functionality. However, as with the basic section, no dynamic mechanisms need to be built on the server side. (Still, only static content and WebApp files (html and js) are placed on the server; no DBMS or dynamic mechanisms ([web services](https://ja.wikipedia.org/wiki/Web%E3%82%B5%E3%83%BC%E3%83%93%E3%82%B9)) are built on the server.)

Furthermore, SVGMap.js can also be used to connect to and operate dynamic servers ([web services](https://ja.wikipedia.org/wiki/Web%E3%82%B5%E3%83%BC%E3%83%93%E3%82%B9)) that use mechanisms for dynamically distributing map and geographic information, such as WMS and WFS, or DBMS . Please refer to [the Web Service Integration section](https://www.svgmap.org/wiki/index.php?title=%E3%83%81%E3%83%A5%E3%83%BC%E3%83%88%E3%83%AA%E3%82%A2%E3%83%AB#.E3.82.A6.E3.82.A7.E3.83.96.E3.82.B5.E3.83.BC.E3.83.93.E3.82.B9.E7.B5.90.E5.90.88.E7.B7.A8).
